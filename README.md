# Spotify для StreamDock (локально)

Управление **локальным** приложением Spotify через CDP — по тому же принципу, что плагин Яндекс Музыки.

## Настройка (один раз)

1. **Закройте Spotify** полностью (включая иконку в трее).
2. В настройках кнопки плагина нажмите **«Запустить Spotify»** — откроется клиент с отладкой.
3. Либо вручную:
   ```
   %APPDATA%\Spotify\Spotify.exe --remote-debugging-port=9223
   ```
4. Порт **9223** по умолчанию.

## Действия

Play/Pause, Next, Previous, Track Info, Like, Volume, Mute, Progress — всё через интерфейс локального Spotify.

## Важно

- Spotify нужно **перезапускать с флагом** после каждого обычного запуска без него.
- Работает только **десктоп Spotify** на этом ПК (не веб и не телефон).
- Версия из Microsoft Store может не подойти — нужен установщик с spotify.com.
